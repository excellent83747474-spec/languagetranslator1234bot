# Language Translator Bot

A powerful Telegram bot for translating text into 100+ languages.

## Features

- 🌍 Translate text to 100+ languages
- 🔍 Auto-detect source language
- 🎯 Custom target language selection
- 📊 Translation confidence scores
- 📖 Translation history
- 📚 Full language list with pagination
- 🎨 User-friendly interface
- 🔒 Privacy-focused
